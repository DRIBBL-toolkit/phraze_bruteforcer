# Phraze Bruteforcer

Phraze clue bruteforcer for SideQuests.

## Installation

Install ts-node and typescript globally:

```bash
  npm install -g typescript ts-node
```

Install all project dependencies with npm

```bash
  npm install
```

## Deployment

First you have to compile the typescript files

```bash
  npm run build
```

Then you can start the built project via

```bash
  npm run build
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PRIVATE_KEY`
